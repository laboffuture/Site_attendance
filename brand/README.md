# Brand assets (source)

Original "Lab of Future" logo art. These are **source files**, not used directly
by the app. The live sidebar logo is the white-text version copied to
`public/img/lof-logo.png` (referenced in `src/views/partials/sidebar.ejs`).

- `LOF_alternate.png` — white-text version (source of the live sidebar logo)
- `LOF_logo.png` — primary logo
- `LOF-Logo.png` — alternate

To change the sidebar logo, replace `public/img/lof-logo.png`.
